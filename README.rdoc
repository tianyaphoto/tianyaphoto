== 欢迎访问基于Rails3.2的图片系统
全文本服务器sunspot启动命令: bundle exec rake sunspot:solr:start 

==完成的功能
* 国际化
* cancan权限控制
* category 增加分类目录
* tag
* 上传图片
* devise 用户认证
* 搜索图片
* 增加用户反馈模块
